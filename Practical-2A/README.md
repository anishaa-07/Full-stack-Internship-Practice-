Build a semantically correct blog page using HTML5 semantic tags.

## Concepts Covered

- header
- nav
- main
- article
- aside
- footer
- table
- time element
- unordered lists

## Learning Outcomes

- Understanding semantic HTML
- Creating structured web pages
- Using tables and lists
- Improving accessibility and readability
